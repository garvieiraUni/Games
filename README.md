# Repositório para projetos de jogos
Esse repositório foi criado para catalogar alguns projetos envolvendo o desenvolvimento de jogos digitais. Os jogos têm como objetivo gerar experiência em programação e principalmente reforçar os conceitos de lógica.

## Jogos:
 ### 1 - Jogo nave:
 É um jogo construído totalmente em C. Tem intuito de ser simples e melhorar o entendimento sobre a linguagem. O jogo roda inteiramente no terminal.

 ### 2 - Pong:
 Jogo simples, porém mais complexo se comparado aos outros. Implementado em python com a biblioteca pygame, o jogo tem como objetivo maior ser experimento para projetos com modelo de IA.

 ### 3 - Jogo da velha:
Famoso jogo da velha, foi construído como uma forma de dominar matrizes em C. O jogo tem três dificuldades e é exibido pelo terminal.
